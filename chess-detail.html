Ce projet de jeu d'échecs est une démonstration de mes compétences en programmation Python et en développement de jeux. Il implémente toutes les règles classiques du jeu d'échecs. Les mouvements spéciaux comme le roque et la prise en sont implementés.

Les pièces suivent les règles officielles du jeu d'échecs, avec un système de validation des mouvements. De plus, les coups possibles d'une pièce s'affichent automatiquement lorque l'on clique dessus. Le jeu comprend aussi une détéction automatique des situations d'échec ainsi que d'échec et mat.

Le jeu dispose d'une interface graphique réalisée avec PyGame. L'interface comprend le plateau de jeu, un bouton pour abandonner, un espace qui donne le score ainsi qu'un endroit qui indique qui doit jouer.
